# Roadmap

## v0.1.0

- Deliver skill workflow and trigger scenarios.
- Publish two scripts: plan generator and theme generator.
- Publish references for design and style presets.
- Provide templates and three concrete use cases.

## v0.1.1

- Refine plan generator into stage-based checklist output.
- Add unified local validation command (`make validate`).
- Add CI validation workflow and PR template.

## v0.1.2

- Align release notes and docs for open-source launch.
- Add changelog and explicit open-source launch checklist.
- Keep CI smoke checks side-effect free.

## v0.2.0

- Expand starter React/Vite showcase template in `assets/templates/starter` (preview delivered in v0.1.x).
- Add scene animation presets with reduced-motion fallback.
- Add quality gates for Lighthouse and responsive breakpoints.

## v0.3.0

- Focus on ecosystem growth and community-driven extensions.

## Growth Milestones

- 30 days: 100+ stars, 20+ forks.
- 60 days: 3 external contributors and 5 showcase submissions.
