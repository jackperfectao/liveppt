# LivePPT v0.2 - 一键 HTML 演示升级

- 目标受众：开源项目维护者 / 产品经理 / 技术决策者
- 主风格：ivory-luxe
- 开始日期：2026-03-29

这是一个最小可运行示例，展示如何把 Markdown 计划直接渲染成可翻页的 HTML 演示页。
你可以把这里的内容替换成自己的项目介绍、发布说明或课程结构。

## 阶段 1：问题定义

很多项目已经有 README、更新日志和截图，但依然缺少一个“几分钟就能讲明白”的演示形态。
当用户只能看到线性文档时，理解成本高，传播效率也会被拉低。

- 任务：明确这次升级的目标不是做更复杂的模板，而是打通 `plan.md -> html`
- 交付：一条命令生成可直接打开的 `dist/index.html`
- 用户收益：非前端用户也能快速产出网页演示

## 阶段 2：最小实现

先用约定格式解析 Markdown 的一级标题、二级标题、段落和列表，避免一上来做重型 DSL。
只要能把现有 `generate_showcase_plan.py` 的输出渲染出来，主链路就算跑通。

- 任务：解析 H1 / H2 / 段落 / 列表
- 交付：独立 HTML 文件，支持翻页、键盘控制、进度条
- 范围边界：先不做复杂卡片布局和多主题编译

## 阶段 3：发布结果

当用户执行完命令后，应该直接获得一个能打开、能分享、能继续人工微调的 HTML 文件。
这会把项目从“规划脚本 + demo 模板”升级为“真正可交付结果”。

- 任务：补 README、示例 plan 和校验脚本
- 交付：3 分钟 Quick Start
- 下一步：再补主题系统、模板映射、多文件静态站导出

## KPI

- 用户能从 Markdown 一条命令生成 HTML
- 输出文件双击即可打开
- README 明确写清能做什么和不能做什么
