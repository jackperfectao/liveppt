# LivePPT GitHub推广动态演示 - 阶段执行清单

- 目标受众：开发者与内容创作者
- 主风格：ivory-luxe
- 开始日期：2026-03-03

## 阶段清单

### 阶段 1：叙事定义
- 任务：定义叙事主线与成功指标，输出最小可行场景图
- 交付：核心叙事一句话 + KPI 列表

### 阶段 2：工程搭建
- 任务：搭建 React + Vite 骨架并接入主题切换机制
- 交付：可运行项目骨架 + 主题切换能力

### 阶段 3：核心场景
- 任务：完成封面、痛点、方案、证据、架构、CTA 六屏
- 交付：可点击浏览的分镜流程

### 阶段 4：交互完善
- 任务：补充核心转场、键盘导航与目录跳转
- 交付：统一的切屏动效与导航体验

### 阶段 5：体验优化
- 任务：做移动端适配与 reduced-motion 降级
- 交付：移动端可用 + 无障碍降级策略

### 阶段 6：发布准备
- 任务：性能优化、发布素材整理与 Demo 复盘
- 交付：可发布 Demo 与复盘要点

## KPI

- Demo 可访问且移动端可用
- Lighthouse Performance >= 85
- Lighthouse Accessibility >= 90
