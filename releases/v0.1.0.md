# Release v0.1.0 - 2026-03-02

首个可开源发布版本，聚焦“点击切换的动态网页演示 + 多风格主题切换”。

## ✨ Added

- 完整 skill 工作流与触发规则（`SKILL.md`）。
- 设计与风格参考文档：
  - `references/design-system-playbook.md`
  - `references/style-presets.md`
- 2 个自动化脚本：
  - `scripts/generate_showcase_plan.py`
  - `scripts/add_theme.py`
- 2 个模板文件：
  - `assets/templates/scene-map.md`
  - `assets/templates/theme-spec.md`
- 3 类示例请求：`examples/use-cases.md`
- 开源基础文档：`README.md`、`ROADMAP.md`、`CONTRIBUTING.md`、`LICENSE`

## ✅ Quality Baseline

- 通过结构检查与脚本 smoke test。
- 执行清单生成脚本与主题生成脚本完成 smoke test。

## 🎯 User Value

- 把传统 PPT 需求快速升级为可交互、可部署的网页演示方案。
- 同一套内容可在多个高级视觉风格间切换，显著提升展示质感与复用效率。
